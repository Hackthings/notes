# CLI

**Run SQL query in CLI postgres:**

```text
psql -U username mydatabase 
mydatabase=#
```

Now you can enter a SQL query, but **do remember to end every query with a semicolon! `;`**

## Read

First, connect to any database: `psql mydb` or `psql -u myuser mydb`. 

`\l` list databases

`\dn` list schemas

`\dt` list tables
